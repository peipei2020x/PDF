# The Sanctuary Archive

> 「ここは外の喧騒が届かない、あなただけの静寂の書庫です」

**The Sanctuary Archive** は、ローカルにあるPDFファイルを没入感のある3D空間で閲覧するための、シングルファイル・電子書庫アプリケーションです。

余計な機能を排除し、暖炉のような微かな灯りと、紙の手触りを感じさせるページめくり体験を提供します。

---

## ✨ 特徴

*   **完全プライベート (Local Execution):**
    *   選択したフォルダ内のPDFは、サーバーに一切アップロードされません。すべての処理はブラウザ内（ローカル）で完結するため、秘匿性の高い文書も安心して閲覧できます。
*   **没入型3D本棚:**
    *   フォルダ内のPDFを自動的に並べ、表紙を生成して美しいグリッド表示にします。
*   **3Dページめくりリーダー:**
    *   物理的な本のようなページめくりアニメーションを実装。
    *   高解像度ディスプレイ（Retina等）に対応した鮮明な文字描画。
    *   **マンガモード:** 右開き（右から左へ読む形式）に最適化されています。
*   **「静寂」のデザイン:**
    *   ダークモードベースの目に優しい配色。
    *   マウスを動かさないとUIが自動的に消え、読書に集中できます。

## 📖 使い方

1.  index.htmlなどを開きます。
2.  **「書庫へ入る」** ボタンをクリックします。
3.  PC内の **PDFが入っているフォルダ** を選択します。
    *   ※ 初回読み込み時、冊数が多い場合は表紙生成に少し時間がかかります。
4.  **操作方法:**

| 画面 | 操作 | アクション |
| :--- | :--- | :--- |
| **本棚** | `矢印キー` | 本の選択移動 |
| | `Enter` / `Click` | 本を開く |
| **読書中** | `←` / `Click(左)` | 次のページへ（右開き想定） |
| | `→` / `Click(右)` | 前のページへ |
| | `Esc` / `Ctrl+Right` | 本を閉じて棚に戻る |

## 🛠️ 技術スタック

このプロジェクトは外部依存を極力減らし、単一のHTMLファイルとして動作するように設計されています。

*   **Core:** HTML5, CSS3 (3D Transforms, CSS Variables), Vanilla JavaScript
*   **PDF Rendering:** [PDF.js](https://mozilla.github.io/pdf.js/) (v3.11.174 via CDN)
*   **Concept:** Single File Application, Privacy-First Design

## ⚠️ 免責事項

*   本ツールはブラウザのローカル環境でのみ動作します。
*   大量のPDF（数百冊以上）を一度に読み込むと、ブラウザのメモリ制限により動作が重くなる場合があります。
*   スマートフォン/タブレットでも動作しますが、PC（デスクトップブラウザ）での体験を推奨します。
